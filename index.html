This is a ecommerce application developed with MEAN stack technology.

Main Features of this application

  • Developed purely with MEAN stack
  • Easy coding pattent
  • Illustrates how to develop complete system with MEAN stack

How to install?

  • Clone it to
  • If you don't have node.js on your computer install it from Here
  • Open command prompt and direct it to mean-app folder
  • run npm install
  • run mongodb with mongod command by directing command prompt on mongodb folder if you have no mongodb install it from Here
  • After installing mongodb direct command prompt to mongodb folder type cd bi and run mondod
  • Mongodb will run and don't close the command prompt
  • Open new command prompt and direct it to mean-app folder
  • run node app Finally browse localhost:3000 on your favorite browser
